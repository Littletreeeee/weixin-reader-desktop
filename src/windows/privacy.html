隐私政策

最后更新日期：2026 年 2 月 1 日

艾特阅读（以下简称"本应用"）是一款桌面客户端软件，旨在为用户提供更好的微信读书阅读体验。我们深知隐私对您的重要性，因此制定本隐私政策向您说明我们如何收集、使用和保护您的信息。

1. 信息收集

本应用在设计上遵循最小化数据收集原则：

  • 本地存储的设置数据：您的阅读偏好设置（如自动翻页间隔、宽屏模式等）仅存储在您的本地设备上。
  • 阅读进度：阅读进度数据由微信读书官方服务器管理，本应用仅作为展示渠道。
  • 不收集个人信息：本应用不主动收集、传输或存储任何个人身份信息、账户信息或阅读内容。

2. 信息使用

本应用收集的有限信息仅用于：

  • 保存您的本地设置偏好，以便在下次启动时恢复
  • 检查应用更新（仅访问 GitHub 发布页面）

3. 第三方服务

本应用会连接以下第三方服务：

  • 微信读书官方网站（weread.qq.com）：用于加载阅读内容，其隐私政策请参阅腾讯公司的相关条款。
  • GitHub：用于检查应用更新和下载更新包。

4. 数据安全

我们采取以下措施保护您的数据安全：

  • 所有设置数据均存储在本地，不上传至任何服务器
  • 与微信读书的通信使用 HTTPS 加密协议
  • 应用更新使用安全签名验证

5. 数据删除

您可以随时通过以下方式删除本应用存储的数据：

  • 卸载本应用将自动删除所有本地存储的设置数据
  • 手动删除应用配置目录下的 settings.json 文件

6. 儿童隐私

本应用不针对 13 岁以下儿童设计或提供服务。如果我们发现无意中收集了儿童的个人信息，将立即删除。

7. 政策变更

我们可能会不时更新本隐私政策。任何变更将在应用内公布并更新"最后更新日期"。继续使用本应用即表示您接受更新后的隐私政策。

8. 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

  • GitHub Issues：https://github.com/dengcb/weixin-reader-desktop/issues
